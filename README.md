# Mercado Libre 
## MeLiChallenge - Test Práctico - Frontend

## Instalación para el cliente y el servidor

```bash
npm install
npm start
```

El cliente corre en http://localhost:3000/
El servidor corre en http://localhost:3001/

## Requerimientos

- React
- React-router-dom
- React hooks 
- TypeScript

- ESLint
- SCSS

- NodeJS + Express
